# ITA-OS Hyprland Profile
# ========================
# This profile uses the default Omarchy Hyprland configuration.
# No additional customization is needed — this IS Omarchy's native experience.
#
# The Hyprland config lives at:
#   ~/.config/hypr/hyprland.conf (user config)
#   ~/.local/share/omarchy/default/hypr/ (omarchy defaults)
#
# When switching to this profile, the profile-switch script will:
# 1. Ensure hyprland is the default session in SDDM
# 2. Remove any KDE autostart entries
# 3. Restore the Omarchy default waybar/hypr configs
